## Project: Unit Tests for a test project
URL: https://stellarburgers.nomoreparties.site/

### Technology Stack
- Java
- JUnit
- Mockito
- Maven
- Jacoco
- Apache Commons Lang (RandomStringUtils)
- Parametrization

### Test Cases
- Test class methods based on each one's logic
- Cover 70% of the main code's methods

### Installation and Execution
1. Clone the repository:  
   *git clone https://github.com/artyomkravts/FinalProject1-UnitTests.git*
2. Navigate to the project folder:  
   *cd <project folder name>*
3. Build the project:  
   *mvn clean install*
4. Run the tests:  
   *mvn clean test*
5. Generate the report:   
   *mvn jacoco:report*   
   (or view the pre-generated report in the site folder > index.html)
